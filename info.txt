🚁 The high-FPS AprilTag method (used in research labs)

1️⃣ Image decimation (the biggest trick)

Instead of running detection on full resolution, the image is downsampled before tag detection.

Example:

Camera:
640×480 → downsample → 160×120

AprilTag has a parameter called quad_decimate that does exactly this.

Effect:

Resolution	FPS
640×480	~15–20
320×240	~40–60
160×120	80–120 FPS

Even developers running AprilTag pipelines mention achieving ~100 FPS by increasing decimation and threading.  

Why it works:
	•	AprilTags are high-contrast binary patterns
	•	They are still detectable even when heavily downsampled.

⸻

2️⃣ Multi-thread detection

AprilTag supports CPU threading:

nthreads = 4

Your Raspberry Pi 4 Model B has 4 Cortex-A72 cores, so detection can run on all cores.

Typical improvement:

Threads	FPS
1	~10
4	~35–50


⸻

3️⃣ Grayscale only (no color)

AprilTag detection converts images to grayscale anyway.

Labs skip color processing entirely.

Pipeline becomes:

Camera → grayscale → tag detection

This saves ~20–30% compute.

⸻

4️⃣ ROI tracking (very powerful)

Once a tag is detected, the next frames only search near the previous location.

Instead of scanning the whole frame.

Example:

Frame size: 640×480
Search region: 200×200

Speed increase: 2–3×

# 5️⃣ Basic Camera + AprilTag Detection Code

Create file:

```
detect_tag.py
```

Code:

```python
import cv2
import numpy as np
from pupil_apriltags import Detector

cap = cv2.VideoCapture(0)

detector = Detector(
    families="tag36h11",
    nthreads=4,
    quad_decimate=3.0,
    quad_sigma=0.0,
    refine_edges=False
)

while True:

    ret, frame = cap.read()
    if not ret:
        break

    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

    results = detector.detect(gray)

    for r in results:

        corners = r.corners.astype(int)

        for i in range(4):
            cv2.line(frame,
                     tuple(corners[i]),
                     tuple(corners[(i+1)%4]),
                     (0,255,0),
                     2)

        cx, cy = int(r.center[0]), int(r.center[1])

        cv2.circle(frame,(cx,cy),5,(0,0,255),-1)

        print("Tag ID:", r.tag_id)

    cv2.imshow("AprilTag Detection",frame)

    if cv2.waitKey(1)==27:
        break

cap.release()
cv2.destroyAllWindows()
```

Run:


You should see:
